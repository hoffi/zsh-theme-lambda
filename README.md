# ZSH-Theme Lambda Î»

A zsh theme that does not overload your terminal.

## Features

* Current path
* Current git informations
* Displays SSH-Session (prefixes "(ssh) <user>@<host>:" to the path)
* Colorizes the Lambda Sign according to the last return status
